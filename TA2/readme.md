# 📘 **README.md - Penjelasan Fitur JavaScript**
Sistem Manajemen Showroom Motor Sport dilengkapi dengan berbagai fitur interaktif yang dibangun menggunakan JavaScript murni. Fitur pertama adalah **Dark Mode** yang memungkinkan pengguna beralih antara tema terang dan gelap dengan preferensi yang tersimpan otomatis di localStorage. Fitur kedua adalah **Wishlist** yang memungkinkan pengguna menyimpan motor favorit ke dalam daftar. Pengguna dapat menambah motor ke wishlist melalui tombol pada setiap card, melihat daftar wishlist di modal, serta menghapus item satu per satu atau mengosongkan seluruh wishlist sekaligus. Data wishlist tersimpan secara permanen di localStorage sehingga tidak hilang meskipun halaman di-refresh.

Fitur ketiga adalah **Sistem Pembelian** dimana setiap tombol "Beli" akan mengurangi stok motor secara real-time dan menonaktifkan tombol jika stok habis. Fitur keempat adalah **Tambah Motor Baru** secara dinamis melalui form input yang dilengkapi validasi data seperti tahun (2015-2024) dan harga minimal Rp 10.000.000. Semua fungsi JavaScript ditulis dengan struktur yang rapi, dilengkapi komentar penjelasan, dan menggunakan manipulasi DOM seperti `querySelector`, `createElement`, serta method array seperti `push`, `splice`, dan `forEach` untuk mengelola data secara efisien.
